Html.

HTML es el lenguaje de texto que nos permite indicar como se desee ordenar el contenido de la página.

Si embargo es importante tener este, pues podemos considerarlo el sistema oseo de cualquier página, sin esta estructura no podríamos diagramar correctamente el contenido del sitio web.

Es importante aclarar al lector que este no es como tal un lenguaje de programación orientado al co mportamiento de almacenamiento o interacción lógica de un programa web, solo es para la interactividad básica del sitio

Debido a su comportamiento de estructura, las etiquetas d e este se separan en partes de extremidades como lo son la cabeza, el pie, entre otros.
