Diseño de Sitios Web

Desarrollar Sitios Web que satisfagan las necesidades del usuario.

Operar lenguajes de Diseño web como HTML5 y CSS3.

Git

Este software es una muy eficaz en su rol, el cual es el de poder llevar un control de versiones. Podemos definir el control de versiones como el control de cambios que se pueden realizar sobre un software específico o un proyecto. En resumen, este software y algunos basados en su comportamiento como lo son Github, nos ayudan a evitar entropía en el desarrollo de un software por parte de todos los involucrados, democratizando la información para evitar malos entendidos y reprocesos en el proyecto a elaborar.

HTML

Este lenguaje de marcado el cual es usado para el desarrollo de páginas web en internet, el cual reduce en sus siglas HyperText Markup Language, que significan en español, Lenguaje de Marcas de Hipertexto. Esto significa que a través de texto se logra estructurar diferentes descripciones o narraciones y vincularlas con múltiples objetos como imágenes, animaciones, entre otros… Por lo que se encarga de dar forma a todo el sitio web y sobre dicha estructura involucrar diferentes scripts y objetos de interacción con el usuario final.

CSS

Este poderoso y eficiente lenguaje de diseño gráfico, de hojas de estilo en cascada o como lo dice su nombre en inglés; Cascading Stylesheets. El cual es la parte a mostrar del sitio web, pues el es el encargado de mantener una perfecta experiencia visual y armonía con lo que ve el usuario de sitio web. Definiendo buenas plantillas de CSS, el desarrollador de la página, puede basarse en un único diseño para evitar la repetición del código reduciendo así la complejidad de desarrollo y el tiempo invertido.
